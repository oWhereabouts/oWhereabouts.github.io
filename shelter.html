A Shelter from the Storm

Created for "Shelter: An Atlas" by Guerilla Cartography, this map was an experimantation with lighting in blender to create a feeling of the arm radiating from huts. To create it I used a raster calculator to combine information to create an approximate weather severity. Next I used a python script to identify coords and colours for hut points and add these into blender.
